# Offline Auto Refresh & Page Monitor Extension

## Overview

This project is a customized browser extension developed for use in secure Network Operations Center (NOC) environments. It is based on an existing open-source extension and has been modified to operate completely offline while maintaining auto-refresh and page monitoring capabilities.

The primary objective of this project is to provide a safe monitoring tool for environments where internet access and external communications are restricted.

---

## Features

* Fully offline operation
* Automatic page refresh at configurable intervals
* Page content monitoring
* Local browser notifications for detected page changes
* No external API calls or cloud dependencies
* Suitable for secure and isolated NOC environments
* Lightweight and easy to deploy

---

## Security

This customized version has been modified with security in mind.

* No internet connectivity required
* No telemetry or analytics
* No external data transmission
* No third-party services
* All monitoring is performed locally within the browser

This makes the extension suitable for use in environments with strict security and compliance requirements.

---
